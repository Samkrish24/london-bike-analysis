# London Bike Sharing Trend Analysis

## Project Overview
Analysis of TfL London bike sharing data to identify usage trends,
peak demand periods, seasonal patterns, and the impact of weather
on ridership. This project simulates a real-world operational
analytics report using Python.

## Business Questions Answered
- Which season has the highest bike demand?
- What time of day are bikes used the most?
- Which days of the week are busiest?
- How does weather affect ridership?

## Tools Used
- Python (Pandas, NumPy)
- Matplotlib & Seaborn (visualisation)
- Jupyter Notebook (VS Code)

## Key Findings
- Summer ridership is significantly higher than winter
- Peak demand occurs at 8am and 5-6pm (commuter driven)
- Weekdays are busier than weekends
- Clear weather produces the highest number of rides

## Dataset
London bike sharing dataset by Hristo Mavrodiev — available on Kaggle.
